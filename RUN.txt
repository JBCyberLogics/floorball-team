Floorball Team - Local Run Guide (Windows)

Prerequisites
1) Node.js 18+ (with npm)
2) MongoDB running locally (one of):
   - MongoDB Windows Service (Community Server)
   - OR Docker: docker run --name local-mongo -p 27017:27017 -d mongo:6.0

Project Structure
floorball-team/
  backend/  (Node/Express API)
  frontend/ (Static HTML/CSS/JS)

1) Install backend dependencies
   PowerShell:
   cd C:\Users\Mr Wanjama\Desktop\pp1\floorball-team\backend
   npm install

2) Environment variables
   Create file: C:\Users\Mr Wanjama\Desktop\pp1\floorball-team\backend\.env
   With contents:
   PORT=5000
   MONGO_URI=mongodb://127.0.0.1:27017/floorball_team
   JWT_SECRET=super_secret_change_me

3) Start MongoDB
   Option A: Windows Service
     - Press Win, type: Services
     - Start service named: MongoDB (if installed)
   Option B: Docker (quickest)
     - Open PowerShell, run:
       docker run --name local-mongo -p 27017:27017 -d mongo:6.0
     - To stop later: docker stop local-mongo

4) Start backend (dev)
   PowerShell:
   cd C:\Users\Mr Wanjama\Desktop\pp1\floorball-team\backend
   npm run dev
   You should see:
   - "Server running on http://localhost:5000"
   - First run will also seed admin: username: admin, password: admin

5) Sanity check API (optional)
   In a new PowerShell window:
   Invoke-RestMethod -Method GET http://localhost:5000/api/health
   Invoke-RestMethod -Method GET http://localhost:5000/api/members
   Invoke-RestMethod -Method GET http://localhost:5000/api/gallery

6) Open frontend
   Option A: Double-click file
     - Open: C:\Users\Mr Wanjama\Desktop\pp1\floorball-team\frontend\index.html
   Option B: VS Code Live Server (recommended)
     - Open the folder in VS Code
     - Install "Live Server" extension
     - Right-click index.html → Open with Live Server

7) Admin login (to manage content)
   - Open: frontend\pages\admin.html
   - Login with:
     username: admin
     password: admin
   - After login:
     - Add member: fill name, position, optional photo → Add
     - Upload media: choose image/video → Upload
     - Delete items using the table actions

8) Verify pages
   - Team page shows members: frontend\pages\team.html
   - Gallery page shows media: frontend\pages\gallery.html

Troubleshooting
• Cannot connect to database / ECONNREFUSED 127.0.0.1:27017
  - Ensure MongoDB is running (service or Docker)
  - Confirm .env has MONGO_URI=mongodb://127.0.0.1:27017/floorball_team

• Health endpoint not responding
  - Ensure backend says "Server running on http://localhost:5000"
  - Check Windows Firewall prompts and allow Node.js

• Uploads not visible
  - Files are saved under backend\uploads
  - Ensure frontend URLs point to http://localhost:5000/uploads/<filename>

• Change default admin password
  - Register a new admin via POST /api/admin/register (temporarily)
  - Or update the seeded account in the database directly
